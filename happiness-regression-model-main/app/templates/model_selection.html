Choose Your Model

SVR

Support Vector Regression (SVR) is a type of supervised learning model used for regression analysis. It works by mapping input data to a high-dimensional feature space so that a linear function can be used for regression.

Linear Regression

Linear Regression is a simple and commonly used statistical technique for modeling the relationship between one or more independent variables and a dependent variable.

Huber

Huber Regression is a robust version of linear regression that is less sensitive to outliers in the data. It combines the advantages of least squares estimation with the robustness of maximum likelihood estimation.

RANSAC

RANdom SAmple Consensus (RANSAC) is an iterative method used to estimate parameters of a mathematical model from a set of observed data points that may contain outliers.

Bayesian Ridge

Bayesian Ridge Regression is a probabilistic model for regression tasks. It places a prior distribution over the coefficients of the regression model, allowing for uncertainty estimates in the predictions.

TheilSen

Theil-Sen Estimator is a robust method for estimating the slope of a linear regression model. It computes the median slope among all pairs of points in the dataset, providing resistance to outliers.

Ridge

Ridge Regression is a technique used to analyze multiple regression data that suffer from multicollinearity. It adds a penalty term to the ordinary least squares objective, shrinking the coefficients and reducing model complexity.

Best Model

The best model is the one that fits your data and problem domain the most accurately. It could be any of the mentioned models or a combination of them, depending on the specific requirements and characteristics of your dataset.
